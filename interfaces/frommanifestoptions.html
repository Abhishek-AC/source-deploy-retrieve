Optional forceAddWildcards

forceAddWildcards: boolean
  • Defined in src/collections/types.ts:64

By default, wildcard members encoutered in the manifest are added to the set e.g. { fullName: '*', type: 'ApexClass' }. If resolveSourcePaths is set, wildcard components are not added to the final set, but are used in the filter when resolving source-backed components to match all components with the wildcard type.

Use this flag to always add wildcard components to the set, regardless of the other conditions.
